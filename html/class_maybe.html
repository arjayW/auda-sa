Detailed Description

template<typename X>
class Maybe< X >

Like a smart pointer, allows for object to not exist (nullptr)

template class Maybe<X> Can be used for monomorphic objects that are stack-allocable, but only conditionally constructed. You might also use it as a member. Initialize with create(), then use like a smart pointer, with *, ->, get(), reset(), or in if()

Like std::optional of C++17 but with other member naming conventions
