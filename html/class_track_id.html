Detailed Description

This is an in-session identifier of track objects across undo states

It does not persist between sessions Default constructed value is not equal to the id of any track that has ever been added to a TrackList, or (directly or transitively) copied from such (A pending additional track that is not yet applied is not considered added) TrackIds are assigned uniquely across projects
