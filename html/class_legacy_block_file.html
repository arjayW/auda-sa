Detailed Description

Audacity 1.1.0 block file format:

  • Header tag: 20 bytes "AudacityBlockFile110"
  • 64K summaries (min, max, RMS, each a 4-byte float)
  • 256 summaries (min, max, RMS, each a 4-byte float)
